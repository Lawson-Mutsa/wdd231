International Space Station Real-Time Tracker

A project demonstrating live API data usage, local storage, and dynamic web content

1. Project Subject

International Space Station Real Time Tracker

2. Project Description

This website will track the real-time location of the International Space Station (ISS) using the Open Notify ISS Location API.

The home page will display the ISS’s current coordinates and a map showing its position over the Earth.

A history page will dynamically list the last 15 recorded positions, including latitude, longitude, and timestamps, stored in local storage.

A modal will allow users to view additional details, such as the ISS’s speed and trajectory information.

The site will consist of three pages:

  • Mission Control (index.html) – Displays the live ISS location, current coordinates, and a map view.
  • Flight Log (flightlog.html) – A dynamic list of the last 15 recorded ISS positions (latitude, longitude, and timestamp), stored in local storage.
  • About the Crew (about.html) – Provides information about the ISS, its mission, and the astronauts currently aboard.

3. Project Purpose

This project demonstrates the use of live data APIs, asynchronous JavaScript, and dynamic content generation to create a functional, real-world-inspired satellite tracker.

The purpose of this project is to build proficiency in handling API data, implementing local storage, and designing a responsive, interactive web tool that educates users about the ISS.

4. User Scenarios

The following scenarios reflect potential users of the ISS Tracker website:

  • Scenario 1: A science teacher wants to show students where the ISS currently is in real-time and explain its mission using live data and maps.
  • Scenario 2: A space enthusiast regularly checks the flight log to review the ISS’s previous orbits and track its speed and trajectory details.
  • Scenario 3: A casual visitor wants to learn more about the astronauts aboard the ISS and the goals of their mission through the About page.

5. Color Schema

This project uses the following primary colors:

  • #00509e — A deep blue used for headings, links, borders, and important highlights to represent the vastness of space.
  • #f0f4f8 — A light blue-gray used as the page background for a clean, subtle sky/space feel.
  • #333333 — Dark gray used for the main body text to ensure readability and contrast.
  • #eeeeee — Light gray background for inline code blocks and subtle UI elements.

6. Typography

The primary font used throughout the website is Segoe UI, a modern, clean sans-serif font known for readability on screens.

  • Body Text: Uses Segoe UI, fallback fonts include Tahoma, Geneva, Verdana, and sans-serif.
  • Headings (h1, h2, h3): Use the same font family, styled with a bold weight and colored deep blue #00509e for emphasis.
  • Code Blocks: Use monospace font for clarity, with a subtle background color #eeeeee.
